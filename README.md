# 🔥 ThermalDAQ-FireSuit

> **An Interactive Thermal Protective Clothing Analysis Platform**
>
> A Python-based application for simulating, analyzing, and visualizing thermal sensor data from fire protective clothing using synthetic datasets, heat flux estimation, burn injury prediction, and an interactive Streamlit dashboard.

## 📖 Overview

ThermalDAQ-FireSuit is an engineering-focused thermal analysis platform inspired by the workflow used in thermal protective clothing evaluation. The project simulates large-scale thermal sensor datasets, processes temperature readings, estimates heat flux, predicts burn severity, and presents the results through an interactive web dashboard.

Since real experimental datasets are often confidential, this project uses a **synthetically generated dataset** designed to resemble thermal sensor measurements for educational and portfolio purposes.

The project demonstrates practical applications of:

* Data Analysis
* Scientific Computing
* Thermal Sensor Simulation
* Interactive Data Visualization
* Engineering Dashboard Development
* Python Software Development

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/dalaladitya635-web/ThermalDAQ-FireSuit.git

cd ThermalDAQ-FireSuit
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

The application will open automatically in your browser.

## 📊 Dataset

The project includes a custom synthetic thermal dataset consisting of:

* **134 virtual thermal sensors**
* **500 time samples**
* More than **67,000 simulated temperature readings**

The dataset simulates:

* Ambient conditions
* Heating phase
* Peak thermal exposure
* Cooling phase
* Random sensor noise

This enables analysis without requiring proprietary or confidential experimental data.
